# Material Families — Interior Zone Academy

An interactive, bilingual (English / Arabic) guide to interior material families
(Marble, Granite, Onyx, and more). Each type shows its photo, properties, and
where it should and shouldn't be used.

This is a **single self-contained HTML file** — all images are embedded and the
only external dependency is Google Fonts (loaded over the web). Nothing else to
install or configure.

## Publish on GitHub Pages

1. Create a new repository on GitHub (Public).
2. Upload **`index.html`** and **`.nojekyll`** to the repository root.
3. Go to **Settings → Pages**.
4. Under **Build and deployment → Source**, choose **Deploy from a branch**.
5. Set **Branch** to `main` and folder to `/ (root)`, then **Save**.
6. Wait ~1 minute. Your site will be live at:
   `https://<your-username>.github.io/<repo-name>/`

## Files

- `index.html` — the tool (open this to view it locally too).
- `.nojekyll` — tells GitHub Pages to serve the file as-is.
- `README.md` — this file (optional; not required for the site to work).
